# Understanding Deep Learning

Esta carpeta contiene cuadernos de Jupyter basados en el libro *Understanding Deep Learning*, de Simon J. D. Prince. Los cuadernos están traducidos al español y combinan explicaciones, visualizaciones, preguntas y ejercicios prácticos en Python.

## Contenido

### 1. Fundamentos

| Cuaderno | Temas principales |
| --- | --- |
| [1. Preliminares matemáticos](1_Preliminares_Matematicos.ipynb) | Ecuaciones lineales, vectores, matrices, funciones especiales y conceptos matemáticos necesarios para el aprendizaje profundo. |
| [2. Aprendizaje supervisado](2_Aprendizaje_Supervisado.ipynb) | Regresión lineal, función de pérdida, visualización de la pérdida y descenso coordenado. |

| [3.2 Redes neuronales superficiales II](3_2_Shallow_Networks_II.ipynb) | Redes con entradas bidimensionales, regiones lineales y múltiples salidas. |
