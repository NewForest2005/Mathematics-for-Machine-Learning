# Mathematics-for-Machine-Learning

This repository contains the course assignments and notebooks on topics covered in *Understanding Deep Learning* by Simon J. D. Prince.

## Understanding Deep Learning homeworks

The `Understanding-Deep-Learning/` directory contains six Python notebook templates for the book's homeworks. Each notebook includes a title, a space for the homework statement, and an empty Python cell for your solution.
