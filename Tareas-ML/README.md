# Tareas

En esta sección se encuentran las tareas del curso.

## Estructura

- **`main.pdf`**: contiene el contenido visualizable de las tareas.
- **`main.tex`**: contiene el código fuente asociado (LaTeX).
- **`archive/`**: contiene el proyecto comprimido en un archivo `.zip`.
- **`figures/`**: contiene las imágenes y demás material multimedia utilizado en las tareas.
